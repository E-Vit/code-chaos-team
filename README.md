# Code Chaos Team Project

## Goal

The project was completed by a team in 8 days to gain practical teamwork
experience and master frontend development tools.

## Key Objectives

- Working with **Vite** bundler
- Team collaboration and **Git** workflow
- Branching and merging experience
- Conflict resolution
- Task planning and management
- Using **Trello** for task tracking

## Tech Acceptance Criteria

- Semantic layout with 3 breakpoints: mobile, tablet, desktop
- **Modern-normalize** included
- Fonts included
- Optimized images in `src/images`
- Favicon added
- HTML & CSS validated
- PageSpeed ≥ 80% for all metrics
- No console errors
- Final version deployed on **GitHub Pages**
- CSS partials in `src/css` imported into `main.css`
- HTML partials in `src/partials` loaded via `<load>`
- Code review completed

## My Contribution

I was responsible for the **Image Gallery** section:

- Adaptive image positioning on screen resize
- Images not only resize but also reposition dynamically
- Modern CSS techniques used for smooth transitions and optimized loading

## Основні завдання

- Робота з бандлером **Vite**

- Оптимізовані зображення у `src/images`
- Додано **фавікон** сторінки

- PageSpeed ≥ 80% для всіх показників
